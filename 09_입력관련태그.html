form 태그/ input 태그

form 태그

        -입력 양식을 작성하는 판(영역, 공간)
        - 내부에 작성된 입력 값을 지정된 곳으로 제출하는 역할

        [속성]
        1) action : 제출될 곳을 지정하는 속성(제출할 주소를 작성)
        2) method(방식) : 설정된 곳으로 데이터를 전달하는 방식
                - 서버에 저장할 것인지, 서버에 저장된 데이터를 호출할 것인지 방식 선택
        3) name : 이름을 지정하는 속성 (JS 사용)
        4) target : 제출된 페이지를 현재/새 탭으로 열기

input 태그

        - 웹 문서에서 사용자(클라이언트)가 입력할 수 있는 양식을 제공하는 태그
        - 많은 타입이 존재하고, input의 변형 형태도 다수존재

        * input 태그에는 name 속성이 필수로 작성되어야 한다.
        (꼭은 아니미잔 작성하는 경우가 80% 이상 많음)

숫자 관련 input 태그

      type="number" : 숫자만 입력 가능한 입력 상자

      type="range"  : 슬라이드바를 이용해서 숫자를 지정
                      (단독 사용 불가, JavaScript  와 혼합해서 사용 가능)

      [속성]
      1. min : 최소값
      2. max : 최대값
      3. step : 숫자가 증가/ 감소하는 간격(단계)
      4. value : 초기 값(기본 값) 지정

날짜/시간 관련 input 태그

radio, checkbox

      - 여러 연관된 값을 묶어서 선택하는 경우에 사용

      - radio : 여러 값 중 하나 선택
      ex) 성별

      - checkbox : 여러 값 중 원하는 모든 것 선택
      ex)상세검색, 관심분야, 취미

      
        연관된 radio, checkbox는 같은 name 속성 값을 가져야 한다.

radio 확인

A팀 B팀 C팀 D팀

checkbox 확인

RED YELLOW ORANGE GREEN

label 태그

      - 입력 관련 태그 (input, select, texarea 등)의 제목을 나타내는 태그

      - 사용법1 : label 태그 내부에 입력 관련 태그 하나 작성하면
                  label 태그를 클릭해도 입력 관련 태그가 선택된다.

      - 사용법2 : 입력 관련 태그의 id 속성 값을 
                  label 태그의 for 속성 값에 똑같이 적으면 연결된다.
